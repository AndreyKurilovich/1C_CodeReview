Задача является сущностью, фиксирующей дефект в исходном коде.

Файл - расположение файла исходного кода.

Описание - текст, который будет выгружен в SonarQube.

Комментарий - свободная информация для внутреннего использования, не выгружается.

Поиск дефекта в файле

Последовательный поиск по строкам - обнаружение каждого условия друг за другом, содержащегося в табличной части. В случае когда все условия выполняются Задача считается присутствующей в файле исходного кода.

Определение начальной и конечной строк поиска задается в поле "Локация в файле".

Вариант поиска Содержит ищет указанное условие в выбранном диапазоне строк, вариант "Не содержит" обратно - ищет отсутствие указанного условия. Вариант поиска "Пустая строка" определяет все ли строки в указанном диапазоне пустые.
